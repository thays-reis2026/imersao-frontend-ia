# Imersão Front-End: IA da Alura + Web Development 🚀

## English
### About the Project
THAFLIX is a dynamic streaming interface developed during the **Alura Front-End Immersion**. Focused on smart UI/UX, accessibility, and dynamic content delivery using HTML, CSS, and JavaScript with AI-assisted workflows.

**Technologies used:**
* HTML5 / CSS3 (Grid & Flexbox)
* JavaScript (ES6+)
* AI-assisted development

---

## Português
### Sobre o Projeto
O THAFLIX é uma interface dinâmica de streaming desenvolvida durante a **Imersão Front-End da Alura**. O projeto foca em interfaces de utilizador inteligentes (UI/UX), acessibilidade e entrega de conteúdo dinâmico, utilizando HTML, CSS e JavaScript com fluxos de trabalho auxiliados por IA.

**Tecnologias utilizadas:**
* HTML5 / CSS3 (Grid & Flexbox)
* JavaScript (ES6+)
* Desenvolvimento auxiliado por IA
